# 💼 Payroll Management System

## 📌 Overview
A Python-based application to manage employee payroll using GUI and database.

## 🎯 Objective
To automate salary calculation and employee data management.

## 🛠️ Technologies
- Python
- Tkinter
- SQLite

## ⚙️ Features
- Add employee details
- Automatic tax calculation (10%)
- Net salary calculation
- Database storage

## ▶️ How to Run
pip install -r requirements.txt  
python main.py

## 📚 Concepts Used
- GUI Programming
- Database Management
- CRUD Operations
